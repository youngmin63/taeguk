# Taeguk - 플로리스트 & 가드닝 웹사이트

Figma 디자인을 기반으로 구현된 Taeguk 플로리스트 및 가드닝 서비스 웹사이트입니다.

## 프로젝트 개요

Taeguk은 가족과 친절함을 바탕으로 한 플로리스트 및 가드닝 서비스 브랜드입니다. 자연스럽고 따뜻한 느낌의 디자인으로 구현되었습니다.

## 주요 기능

- **서비스 소개**: 계절별 꽃다발, 인테리어 스타일링, 가든 스타일링, 이벤트 설치, 개인 상담
- **프로젝트 스토리**: 실제 작업 사례들을 카드 형태로 소개
- **고객 후기**: 실제 고객들의 생생한 후기
- **About Taeguk**: 브랜드 스토리와 철학 소개
- **Instagram 갤러리**: 작업 사진들을 갤러리 형태로 전시

## 기술 스택

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Font**: Cormorant Upright (Google Fonts)

## 프로젝트 구조

```
src/
├── components/
│   ├── ServiceCard.tsx      # 서비스 카드 컴포넌트
│   ├── StoryCard.tsx       # 스토리 카드 컴포넌트
│   └── TestimonialCard.tsx # 고객 후기 카드 컴포넌트
├── App.tsx                 # 메인 앱 컴포넌트
├── main.tsx               # 앱 진입점
└── index.css              # 글로벌 스타일
```

## 실행 방법

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:5173` 접속
